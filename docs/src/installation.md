# Installation

## Prerequisites

- [Rust and Cargo](https://www.rust-lang.org/tools/install)

## Install from crates.io

```bash
cargo install ripcalc
```

## Install from source

```bash
git clone <repository-url>
cd ripcalc
cargo install --path .
```

## Build from source

```bash
cargo build --release
```

The binary will be available at `target/release/ripcalc`.
